📌 Project Name: OmniSight AI - Smart Analytics Dashboard
I have developed OmniSight AI, a high-performance, smart dashboard designed for instant data visualization and processing. This tool empowers users to transform raw Excel or CSV files into actionable, interactive insights within seconds, directly in the browser.

🛠️ Tech Stack:
HTML5 & Tailwind CSS: Crafted with a modern Glassmorphism aesthetic to ensure a professional and seamless user experience.

JavaScript (ES6+): The core engine that handles dynamic data processing without the need for a backend.

Chart.js: Utilized to render 8 interactive chart types (Bar, Line, Pie, Radar, etc.) featuring customized tooltips for deep-dive analysis.

XLSX Library: Integrated to enable the browser to parse and process Excel/CSV files instantly.

📂 Project Architecture & Sections:
Dashboard Tab (Visual Analytics):

Displays real-time Key Performance Indicators (KPIs) such as Total Sum, Average, and Row Counts.

Features 8 synchronized charts optimized for clarity by removing distracting legends and focusing on interactive data points.
